# vscode-consolelog-snippet README

This is a really simple snippet for vscode. I wanna print both the name and the value of parameter, eg: console.log('param', param).

clg -> Tab -> param -> Tab -> DONE!

## Features

| Snippet | Purpose         |
| ------- | --------------- |
| `clg`   | `console.log()` |
